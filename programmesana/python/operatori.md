# Operatori
Aritmētiskie operatori

| Operators | Apraksts    | Piemērs  |
|:-----|:------------|:------------|
| +   | Saskaitīt     |  4 + 4        |
| -   | Atņemšana | 4  - 3     |
| *    | Reizināšana     | 4 * 3|
| /   | Dalīšana      | 4 / 3 |
| %    | Modulis       | 4 < 5 |
| **   | Kāpināšana | 4**3 |
| //   | Dalīšana iegūstot veselu skaitli    | 13 // 4 |

Vairāk par Python operātoriem var lasīt [šeit](https://www.w3schools.com/python/python_operators.asp).
